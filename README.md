# Voting Machine

## Features
Normal Mode: Voting for 4 candidates / Reset  
Winner Mode: Display winner / Reset  
If the power is cut out the votes are still stored in the EEPROM and can be recovered  

## Drivers
GPIO  
Interrupt  
EEPROM  
LCD  

## Components
Microcontroller(PIC18f4620)  
LCD(LM016L)  
Push Buttons  
Resistors
